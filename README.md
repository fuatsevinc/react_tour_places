# react_tour_places – React Tour Places App

> A React application for browsing and discovering tour destinations and travel places.

---

## About the Project

react_tour_places is a React-based web application that showcases a collection of travel destinations and tour places. Users can browse through different tour locations, each presented with relevant details. The project is styled with SCSS for a clean and responsive design.

---

## Features

- Browse and discover tour destinations
- Interactive cards for each travel place
- Responsive design with SCSS styling
- React component-based architecture
- Clean and modern UI

## Getting Started

Follow these steps to run the project locally:

1. **Clone the repository**
   ```bash
   git clone https://github.com/fuatsevinc/react_tour_places
   cd react_tour_places
   ```

2. **Install dependencies**
   ```bash
   yarn install
   ```

3. **Start the development server**
   ```bash
   yarn start
   ```

4. Open your browser and visit: `http://localhost:3000`
